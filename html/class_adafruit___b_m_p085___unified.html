Absolute altitude

Calculates the altitude (in meters) from the specified atmospheric pressure (in hPa), and sea-level pressure (in hPa). Note that this function just calls the overload of pressureToAltitude which takes seaLevel and atmospheric pressure–temperature is ignored. The original implementation of this function was based on calculations from Wikipedia which are not accurate at higher altitudes. To keep compatibility with old code this function remains with the same interface, but it calls the more accurate calculation.
